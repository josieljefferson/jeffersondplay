# 📺 IPTV System

Sistema completo de IPTV com atualização automática via GitHub Actions e suporte a Xtream Codes.

## 🚀 Características

- ✅ Atualização automática a cada 6 horas via GitHub Actions
- ✅ Suporte completo a Xtream Codes
- ✅ Web player integrado
- ✅ API RESTful completa
- ✅ Processamento de múltiplas listas M3U
- ✅ Remoção automática de URLs duplicadas
- ✅ Suporte a EPG
- ✅ Interface web responsiva
- ✅ Hospedagem gratuita no Render
